# javascript-web-alpinejs-counter

## Description
A demo of alpinejs using a clickable counter.

## Tech stack
- alpinejs

## Docker stack
- httpd:latest

## To run
`sudo ./install.sh -u`
- [Available at](http://localhost)

## To stop
`sudo ./install.sh -d`

## To see help
`sudo ./install.sh -h`
